HTML Tags

HTML tags are like keywords which define how the web browser will format and display the content. Tags allow a browser to distinguish between html content and simple content. The following are examples of five different HTML tags.

Header Tag
Acts as the master header of the page or the header of an article depending on parents
Navigation Tag
The navigation of the page, will contain the links to other pages/parts of the website
Article Tag
The tag used for containing independent piece of content. It could be shared as stand alone piece of content and will make sense.
Aside Tag
Secondary content on the page, this information is not part of the main topic of the page and is not required.
Section Tag
Group of related content on the page. A section must have its own heading
